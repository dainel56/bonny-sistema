⚙ Instalación
Cómo configurar este sistema en GitHub Pages — permanente y gratuito
Pasos de instalación
1
Descarga este archivo HTML
Guarda este archivo como index.html en tu computadora. El nombre debe ser exactamente ese.
2
Crea un repositorio en GitHub
Ve a github.com → New → Nombre: bonny-sistema → Public → Add README → Create repository
3
Sube el archivo index.html
Dentro del repositorio → Add file → Upload files → arrastra el index.html → Commit changes
4
Activa GitHub Pages
Settings → Pages → Branch: main → / (root) → Save. Espera 2-3 minutos. Tu URL será: https://tuusuario.github.io/bonny-sistema
5
Configura el Proyecto Bonny en Claude.ai
En Claude.ai → Projects → Bonny Finance → Instructions. El sistema de producción v4.0 debe estar como instrucciones del proyecto para que los prompts funcionen correctamente.
6
Flujo de trabajo diario
Abre este sistema en tu URL de GitHub → genera el prompt → cópialo con 1 clic → pégalo en Claude.ai (plan Pro que ya tienes) → copia el resultado → listo.

Flujo completo automatizado v5.0
1
Configura tu API Key de Gemini
Obtenla en aistudio.google.com → Get API Key. Pégala en el campo del menú lateral. Se guarda solo en tu navegador (localStorage), nunca en GitHub.
2
Ingresa el tema y genera el JSON Maestro
Gemini Flash genera el guion completo + 54 prompts de imagen + timestamps. Todo en formato JSON listo para Python.
3
Exporta el JSON a tu PC
Haz clic en "Exportar proyecto_video.json". El archivo se descarga a tu carpeta de Descargas.
4
Ejecuta el Orquestador Python
Python detecta el JSON, genera el audio con edge-tts, descarga las imágenes de Imagen 4 (respetando 15 RPM), y ensambla el video final con MoviePy.
5
Recibe el video en /output/
Versión limpia + versión con subtítulos quemados + 4 Shorts individuales. Solo necesitas hacer cortes finales en CapCut.
